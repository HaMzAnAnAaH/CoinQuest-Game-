# CoinQuest 🎮

A lightweight 2D platformer game where the player jumps across platforms, collects coins, and avoids enemies to win.

## 🚀 Features

* Player movement and jumping mechanics
* Coin collection system
* Enemy collision (lose condition)
* Win condition upon collecting all coins

## 🛠️ Technologies

* C++
* Basic game development concepts (game loop, collision detection)

## 🎯 Purpose

This project was built to practice and demonstrate fundamental programming and game development skills.

---

Simple project, but a strong step toward building more advanced games.
